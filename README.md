# Vip Video Player

VIP视频解析-前端页面，解析功能由第三方接口实现，本项目无后台，不收集任何数据。

此项目仅用于Web开发学习、交流，请勿商用。

---

## 运行环境

纯静态页面，浏览器打开即用，无需PHP运行环境。

总体积小于 0.1MB。

CodePen 版本在这里 ✏️ [codepen.io](https://codepen.io/jiangmenghao/pen/gOrrQGQ).
